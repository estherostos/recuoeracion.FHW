Bitdefender para Windows

Para instalarlo nos dirigiremos a la página oficial

PULSAR AQUI

1.) Pase el raton "Para el hogar" y sigue al apartado del sistema operativo al que quieres instalar el antivirus, después desplaza hacia la seccion de "Antivirus free"


2.) Haz clik en "Descargar ahora" y espera un rato hasta que se descargue el archivo .exe, despues dale doble click para empezar la instalación


3.) Cuando has terminado la instalacion, nos pedira reiniciar el equipo y posteriormente crear una cuenta o iniciar sesion si ya tienes una


4.) Al final te pedira introducir un codigo de activacion para la versión de pago o octar por por la version gratis




ClamAV para Ubuntu

1.) Los primero sera abrir el terminal de Ubuntu y escribir comando [sudo apt install clamav]


2.) Después tienes que volver a introducir otro comando [sudo apt install clamtk] que se trata de la interfaz gráfica del antivirus
